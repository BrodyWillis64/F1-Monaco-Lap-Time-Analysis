# F1-Monaco-Lap-Time-Analysis
This is a project conducted as part of a coursework for my Time Series module at Queen Mary University of London. I have a time series representing the fastest Formula 1 lap times from every single Monaco Grand Prix held.

The purpose of this project is to utilise Meta’s Prophet forecasting system on the lap time data to create a forecast and to try and predict how fast the lap times will be in the future, focusing on details like overall trends and variability of the time series and comment on potential reasons or causes of this.
